1. MBee.h (папка includes) - описание класса MBee, базового для всех поддерживаемых проектов и классов, определяющих адресацию в сети.
 - базовые классы для всех поддерживаемых проектов.
2. MBee.cpp - определение открытых методов класса MBee.
3. System.h (папка includes) - объявления для вспомогательных классов и функций, реализация части которых является платформозависимой. 
4. System.cpp - определения методов вспомогательных классов и функций, не зависящих от аппаратной платформы.
5. SerialStar.h (папка includes) - объявление производных классов MBee, а также специфических констант и классов для поддержки программного обеспечения SerialStar.
6. SerialStar.cpp - определения методов классов, поддерживающих проект SerialStar.
7. WindowsSerial.cpp - определения методов вспомогательных классов, объявленных в файле System.h, реализация которых привязана к платформе Windows.
8. LinuxSerial.cpp - определения методов вспомогательных классов, объявленных в файле System.h, реализация которых привязана к платформе Linux.
9. MSP430Serial.cpp - определения методов вспомогательных классов, объявленных в файле System.h, и предназначенных для работы на микроконтроллерах семейства MSP430, производства Texas Instruments.
10. STM32Serial.cpp - определения методов вспомогательных классов, объявленных в файле System.h, и предназначенных для работы на микроконтроллерах семейства STM32, производства STmicroelectronics.
11. compile.bat - командный файл для компиляции файла приложения, основанного на библиотеке MBee C++, в среде Windows. При его вызове в качестве первого параметра ему должно быть передано имя исходного файла.
   Второй параметр - имя выходного exe файла без расширения.
12. compile.sh - файл, аналогичный по назначению файлу compile.bat, но предназначенный для Linux-платформ.
13. Папка examples содержит примеры исходных файлов, использующих библиотеку MBee C++.
14. Папка docs содержит документацию по библиотеке (файл /html/index.html).
