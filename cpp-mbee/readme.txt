Проект предназначен для поддержки программного обеспечения беспроводных радиомодулей MBee, разрабатываемых и производимых фирмой "Системы, модули и компоненты".

Версия компилятора для Windows, использованного при разработке библиотеки: g++ (i686-posix-dwarf-rev0, Built by MinGW-W64 project) 8.1.0.
Версия компилятора для Linux (Raspbian Stretch), использованного при разработке библиотеки: g++ (Raspbian 6.3.0-18+rpi1+deb9u1) 6.3.0 20170516.
Компилятор для микроконтроллеров MSP430 IAR C/C++ Compiler for MSP430 7.12.1 (7.12.1.987).
Компилятор для микроконтроллеров MSP430 TI v18.12.2.LTS (Code Composer Studio 9.1.0.00010).

Поддерживаемое ПО:
1. SeralStar (http://sysmc.ru/documentation/sw_SerialStar.pdf).

Документация на библиотеку находится в cpp-mbee\docs\html\index.html.
Примеры располагаются в cpp-mbee\examples в каталогах, соответствующих целевым платформам. Внутри этих папок могут находиться отдельные redme-файлы, содержащие доплнительную информацию. Как правило, комментариев, содержащихся в исходных текстах примеров бывает
достаточно для понимания принципов работы с библиотекой. Для удобства пользователей в ряде случаев в каталоге примеров в docs\html\index.html имеется подробное описание конкретного примера.

Для ускорения изучения библиотеки и начала создания собственных проектов, рекомендуется в начале работы протестировать имеющиеся примеры. Впоследствии ими можно пользоваться как готовыми шаблонами при создании собственных приложений.
