◆ setPayloadLength()

void setPayloadLength ( uint8_t  payloadLength)

Устанавливает число байт, которое надо поместить в поле данных отправляемого пакета. Например, если буфер, на который указывает указатель, установленный функцией setPayload(), содержит 40 байт, а требуется передать только первые 5, то вызывается эта функция с параметром 5. Значение параметра не должно превышать длину буфера.
