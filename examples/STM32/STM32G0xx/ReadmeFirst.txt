Папка Projects содержит примеры, предназначенные для демонстрации приемов работы с библиотекой cpp-mbee при разработке программного обеспечения для микроконтроллеров семейства STM32.
Структура и состав каталогов полностью совпадает с примерами, входящими в репозиторий для каждого семейства микроконтроллеров STM32.
Индивидуальное описание каждого примера находится в файле readme.txt в папке с исходными кодами примера. Подробные комментарии, имеющиеся в исходных файлах примеров, совместимы с системой автоматического документирования Doxygen.

Настройка аппаратных средств, задействованных в примерах, осуществляется с помощью SMT32CubeMX, проект для которого имеет название, совпадающее с названием примера. Работа с аппаратными ресурсами микроконтроллера организована на уровне HAL.
Для всех примеров доступны 2 конфигурации - Debug(отладочная), и Release.
